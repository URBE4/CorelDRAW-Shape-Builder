Порядок работ по проекту Shape Builder (идём сверху вниз)
============================================================
Целевая среда: CorelDRAW Graphics Suite 2026 (Programs64\CorelDRW.exe; COM ProgID .28/.27… — см. ShapeBuilder_CorelConnect.ps1).
Скрипты deploy_all.ps1 / deploy_direct.ps1 рассчитаны на 2026; при установке в нестандартную папку — SHAPE_CORELDRW_EXE или -CorelExe.

1. Стабильность и обновление
   - Актуальный релиз: 0.1.4 (2026), git tag v0.1.4; см. RELEASE_0_1_4_2026.txt.
   - Источник правды: .bas + UserForm1_Code.txt в папке проекта.
   - deploy_direct.ps1 → затем ОБЯЗАТЕЛЬНО VBA: Сохранить GlobalMacros (COM Save в Corel часто недоступен).
   - Копии после deploy: vba_backup\, vba_export_last\ — см. RESTORE.txt.
   - Запуск: только Module1.Фасады (Module2 удаляется скриптом; прочие входы — Private).
   - Проверка версии: заголовок формы v… [2026-one-facades] или Immediate: ?Module1.MEBEL_MACRO_VERSION(0)

2. Раскрой (Module1)
   - Алгоритмы: nesting / Guillotine / строки.
   - Лист В×Ш, зазор, поворот, волокно, запас листов.

3. ЧПУ
   - Галочка CUT/TEXT, СОЗДАТЬ, затем DXF.

4. Интерфейс (UserForm1)
   - Панель «Контур» в одну строку (как в Corel) — 0.1.3.
   - Вкладки, цвета, подсказки, список, 3D-превью.
   - Corel MSForms (438): без BackColor у CheckBox/Page; без BackColor у ComboBox;
     без BorderColor у ListBox; без TakeFocusOnClick, Font.Bold на кнопках;
     без ControlTipText; без Label.WordWrap; ComboBox/текст — по возможности стандартные.

5. Данные
   - CSV, last_facades.csv, реестр настроек.

6. Документация
   - RESTORE.txt, OPERATORS_RU.txt при необходимости.
